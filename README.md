# Customer Behavior Analysis Dashboard

## 📊 Overview
This project analyzes customer shopping behavior using SQL, Python, and Power BI.

## 🛠 Tools Used
- MySQL
- Python (Pandas)
- Power BI

## ❓ Business Questions
- Which product categories generate the highest revenue?
- Which customer segments spend the most?
- Do subscribed customers spend more?
- What factors impact customer satisfaction?

## 📈 Key Insights
- Child segment generates highest revenue
- Non-subscribed customers contribute more total revenue
- Revenue is evenly distributed across seasons
